=== Exercise 6.13 ===
Yes. Take the cyclic permutation matrix on n states: P(i+1,i)=1 and P(1,n)=1.
Then P^k(1,2) = 0 for k=1,...,n-1 but P^n = I so P^n(1,2) = 0 too... wait.
Actually: P^k(i,j)=1 iff j -> i in exactly k steps, i.e., iff k = i-j (mod n).
For entry (2,1): P^k(2,1)=0 for k=1,...,n-1, and P^n(2,1)=0 as well (since P^n=I).
So P^1(2,1)=1: this entry is nonzero at k=1. Not the example we want.
Try entry (1,2): P^k(1,2)=1 iff k = n-1 (mod n). So (1,2) is 0 for k=1,...,n-2,
and nonzero at k=n-1. Yes! For the n-cycle, T^k(1,2)=0 for k=1,...,n-2, nonzero at k=n-1.
Example for n=5: T^k(1,2)=0 for k=1,2,3, nonzero at k=4 (=n-1):
  k=1: T^k(1,2)=0
  k=2: T^k(1,2)=0
  k=3: T^k(1,2)=0
  k=4: T^k(1,2)=1
  k=5: T^k(1,2)=0

=== Exercise 6.14 ===

(b) Steady state: solve T*x=x with x1+x2=1:
    (alpha-1)*x1 + (1-beta)*x2 = 0 => x1/x2 = (1-beta)/(1-alpha)

=== Exercise 6.15 ===
Claim: If T^k has all positive entries, then T^j for j > k also has all positive entries.
Proof: Write j = k + m for some m >= 1. Then T^j = T^m * T^k.
Since T^k > 0 (entrywise) and T^m >= 0 (nonneg) with column sums = 1 (at least one
positive entry per column), the product T^m * T^k has all positive entries.
More explicitly: (T^j)_{ij} = sum_l (T^m)_{il} * (T^k)_{lj}.
Each term (T^k)_{lj} > 0, so the sum is positive as long as (T^m)_{il} > 0 for some l.
Since T is a transition matrix, at least one entry per column is positive, hence T^m has
at least one positive entry per column, giving (T^j)_{ij} > 0.
