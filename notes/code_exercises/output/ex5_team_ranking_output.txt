=== Exercise 5.4 ===
Before replacing final row, M = A^T*A.
The diagonal entry m_{ii} = number of games team i played.
Sum of diagonal = sum of all teams' game counts = 2 * (total games).
Since each game involves exactly 2 teams, it is counted twice.
Hence the sum of diagonal = 2*G (where G = total games) => always even.

Example: 2 teams, T1 beats T2 by k. Then r1-r2=k.
The matrix equation A*r = p is just [1,-1]*[r1;r2] = [k].
This is ONE equation with two unknowns => infinitely many solutions.
After Massey modification: M=[1,-1;1,1], q=[k;0] => unique solution.
No LS needed since M is now invertible.
